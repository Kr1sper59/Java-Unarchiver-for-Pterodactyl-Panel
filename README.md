# Java Unarchiver for Pterodactyl Panel

В панели существуют ограничения на распаковку архивов через встроенный файловый менеджер, что может затруднять перенос крупных сборок.

Этот проект решает проблему, предоставляя JAR-утилиту для распаковки архивов прямо при запуске сервера.

---

## Возможности
- Поддержка `.zip`
- Поддержка `.tar.gz`
- Удаление архива после распаковки

---

## Установка

1. Скачайте последний билд из [Releases](https://github.com/Kr1sper59/Java-Unarchiver-for-Pterodactyl-Panel/releases)
2. Загрузите на сервер
3. Установите `server.jar` (или другое имя ядра в настройках запуска)
4. Убедитесь, что используется Java 21
5. Поместите архив рядом с JAR-файлом
6. Создайте `config.yml` по шаблону ниже
7. Запустите сервер

---

## Конфигурация
```yaml
archive: archive_name.zip
deleteAfter: false
```

### Параметры:
archive - имя архива (.zip или .tar.gz)
deleteAfter - удалить архив после распаковки

⚠️ Конфиг не создаётся автоматически - создайте его вручную.
